Tag guide (WIP)

On the GPTChats on this site and #chambers on my Discord server, I want to introduce tags to conversations. These are unique keywords that show you what topics or features a conversation is going to have, and you can highlight conversations involving a particular tag by using a search feature, either available on the site, by pressing Ctrl/Cmd+F, or using another native search feature, and typing strings in the search bar.

All of them have a "t" prefix so you can search for keywords by the tags specifically.
