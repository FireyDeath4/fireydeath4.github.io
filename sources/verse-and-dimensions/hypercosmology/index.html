Hypercosmology

Hypercosmology is a specific branch of cosmology that considers hypercosmological objects. An object is usually considered hypercosmological if it is independent of some logically or mathematically consistent hierarchy. Typically objects that hold some property regardless of other objects' definitions that contradict its own, including all Box-equivalent structures, are considered hypercosmological.
